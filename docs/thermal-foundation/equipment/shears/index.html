Shears are tools with specific uses in Minecraft. They can be used to obtain Wool from Sheep, harvest plants and Cobwebs, and to disarm Tripwire traps. Thermal Foundation adds Shears types for all of its base metals and some of its alloys.

Shears can be repaired with an Anvil, using the metals they are crafted with.

Crafting

  • 2x Metal Ingot

Enchantments

Shears can have the following enchantments applied to them, but only by using an Anvil:

  • Efficiency
  • Unbreaking
  • Silk Touch
