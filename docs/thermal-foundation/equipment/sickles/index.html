--- title: Sickles --- {% include tftoolimages.html type='sickle' %}

Sickles are tools that can be used for harvesting crops quickly. When a plant-type block is broken using a Sickle, all plant-type blocks in a 7-by-7-block area are broken as well. Thermal Foundation adds Sickle types for all of its base metals and some of its alloys.

Sickles can be repaired with an Anvil, using the metals they are crafted with.

Crafting


Shaped Crafting
  • 3x Metal Ingot
  • Stick

Enchantments

Shovels can have the following enchantments applied to them:

  • Efficiency
  • Unbreaking
  • Silk Touch*
  • Fortune*
*: A Sickle cannot have both Silk Touch and Fortune on it.
