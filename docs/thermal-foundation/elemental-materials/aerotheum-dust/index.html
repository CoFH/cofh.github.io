Aerotheum Dust is a type of dust added by Thermal Foundation. It is created from Redstone, Sand, Niter and Blitz Powder. It is the air elemental material.

Crafting


Shapeless Crafting
  • Sand
  • Niter
  • Redstone
  • Blitz Powder

Usage

Aerotheum Dust has only one use: it can be molten down into Zephyrean Aerotheum. It doesn't have any other uses, but it is planned to get some in later releases.
