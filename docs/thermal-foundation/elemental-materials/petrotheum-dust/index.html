Petrotheum Dust is a type of dust added by Thermal Foundation. It is created from Redstone, Clay, Pulverized Obsidian and Basalz Powder. It is the earth elemental material.

Crafting


Shapeless Crafting
  • Clay
  • Pulverized Obsidian
  • Redstone
  • Basalz Powder

Usage

Petrotheum Dust has only one use: it can be molten down into Tectonic Petrotheum. It doesn't have any other uses, but it is planned to get some in later releases.
