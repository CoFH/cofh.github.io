Ferrous Metal is one of the base metals added by Thermal Foundation. It can be obtained from Ferrous Ore and, less commonly, Iron Ore.

Usage

Like most base metals in the mod, Ferrous Metal can be used to craft tools and armor. Other than that, it does not serve much purpose, besides being a critical component in creating the much more useful alloy Invar.

Naming

The reason the metal is not called Nickel is because it is not part of the seven metals of antiquity. Steve? doesn't know what to call the metal, resulting in the less 'logical' name Ferrous Metal.
