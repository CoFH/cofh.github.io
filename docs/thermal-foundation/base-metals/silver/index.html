Silver is one of the base metals added by Thermal Foundation. It can be obtained from Silver Ore and, less commonly, Lead Ore.

Usage

Like most base metals in the mod, Silver can be used to craft tools and armor. It also has various uses as a generic crafting ingredient in Thermal Expansion. More importantly, Silver is a component in various alloys: Electrum, Signalum, Lumium and Enderium all require different amounts of Silver to create.
