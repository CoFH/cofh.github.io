Tin is one of the base metals added by Thermal Foundation. It can be obtained from Tin Ore. Like Copper, the metal is very common in modded Minecraft.

Tin Ore can also be processed in various ways in an Induction Smelter, with varying byproducts.

Usage

Like most base metals in the mod, Tin can be used to craft tools and armor. It is also a common ingredient for creating basic devices in Thermal Expansion. Furthermore, Tin can be combined with Copper to create Bronze. It can also be combined with Silver and Energized Glowstone to create Lumium, or combined with Silver, Shiny Metal and Resonant Ender to create Enderium.
