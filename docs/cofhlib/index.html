--- title: CoFHLib redirect_from: /mods/cofhlib/ ---

CoFHLib, also known as the CoFH Common Library, is a libary that allows modders to develop their mods more easily. CoFHLib provides a lot of the 'boring', yet essential code for creating mods. It is not actually a mod in itself, though.

CoFHLib also contains the CoFH API, which may be used by other mods to integrate better with the various Team CoFH mods.

Note: CoFH Core includes CoFHLib by default.
