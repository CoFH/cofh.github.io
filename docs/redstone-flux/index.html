Redstone Flux, often referred to as RF, is the energy system used by the mods of Team CoFH. As described in Redstone Flux: A Primer, it is a natural extension of Minecraft's Redstone power system.

Various blocks and items in the CoFH mods consume Redstone Flux energy to accomplish all kinds of tasks. Redstone Flux is usually consumed for one of two reasons:

  • replacing the requirement of manual labor in automated systems;
  • adding a cost to do things that otherwise wouldn't be possible.

The CoFH mods offer various ways to generate, move and store Redstone Flux, such as Thermal Expansion's Dynamos and Energy Cell, and Thermal Dynamics's Fluxducts. However, many other mods add more ways to handle Redstone Flux, in addition to more devices that consume it.

Development

Redstone Flux has a simple, easy-to-implement API that mods may use to add compatibility for it. This API is part of the CoFH API in CoFHLib, but a standalone version can also be found here.

Many mods are compatible with the Redstone Flux system, and thus with the CoFH mods and each other. However, if you are the developer of a mod that involves energy in some way, please don't feel pressured to make it compatible with Redstone Flux. See this post for more on that.
