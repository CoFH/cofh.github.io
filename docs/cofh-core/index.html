--- title: CoFH Core redirect_from: /mods/cofh-core/ ---

CoFH Core is a mod that must be installed in order for most Team CoFH mods to run. The mod extends upon CoFHLib, providing common functionality specifically for the mods of Team CoFH.

Other than providing common functionality for the other mods, CoFH Core also provides a set of useful commands, a friend list, some configurable game tweaks, and a system that allows users to customize parts of world generation. It also notifies players of updates for the Team CoFH mods.
