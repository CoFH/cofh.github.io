A **reusable morb** is a [morb](/docs/morb/) that never breaks.

Usage
-----

A reusable morb works like a regular [morb](/docs/morb/), except it doesn't
break after spawning a [mob](https://minecraft.gamepedia.com/Mob) of the
contained type.
