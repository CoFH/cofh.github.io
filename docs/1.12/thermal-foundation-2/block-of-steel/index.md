**Blocks of steel** are blocks made from [steel ingots](/docs/1.12/thermal-foundation-2/steel-ingot/).


Obtaining
---------

Blocks of steel can only be mined with an [iron
pickaxe](https://minecraft.gamepedia.com/Pickaxe) or better.

Usage
-----

### Storage
Blocks of steel can be used to store [steel ingots](/docs/1.12/thermal-foundation-2/steel-ingot/) in a
compact way.

### Mob spawning
Mobs will not spawn on top of blocks of steel.

### Beacons
Blocks of steel can be used to build
[beacon](https://minecraft.gamepedia.com/Beacon) pyramids.
