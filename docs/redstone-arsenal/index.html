--- title: Redstone Arsenal redirect_from: /mods/redstone-arsenal/ ---

Redstone Arsenal is a mod in the 'Thermal' series that contains a powerful set of tools. It works on its own, but for the best experience, it should be used together with Thermal Expansion. The mod showcases the implementation of the Redstone Flux API.

Fluxed Equipment
A complete set of RF-powered tools and armor with unique abilities!
Fluxed Blocks
Be hurt and charge your items! What better combination is there?
