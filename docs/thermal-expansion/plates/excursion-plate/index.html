--- title: Excursion Plate ---

The Excursion Plate is one of the Plates added by Thermal Expansion. It is an upgraded version of the Impulse Plate. It uses Redstone Flux to project a beam of force which moves entities in a certain direction, defying gravity. The beam has a configurable maximum length, and dissipates when far enough from the Plate.

The Plate can be placed on any side of a block. Its beam normally moves entities away from the Plate, but it can be configured to pull entities towards itself instead, or to switch between the two according to a Redstone signal.

Crafting


Shaped Crafting
  • Impulse Plate
  • Redstone Reception Coil
  • 3x Block of Quartz
  • 4x Signalum Ingot

Power

The Excursion Plate consumes energy whenever possible to project its beam. The amount of RF/t consumed is equal to the length in blocks of the Plate's beam; for example, the Plate uses 20 RF/t when the projected beam is 20 blocks long. The Plate can store 200000 RF internally.

GUI

The Excursion Plate can be configured by sneaking and using a Multimeter on it. The Plate's GUI is then opened:

1. Energy Gauge
Displays how much Redstone Flux is currently stored in the Excursion Plate.
2. Distance Controls
Change the length in blocks of the Excursion Plate's beam. Can be anywhere between 1 and 26 blocks.
3. Information Tab
Displays a bit of information about the Excursion Plate.
4. Redstone Control Tab
Allows setting how the Excursion Plate responds to Redstone signals. Note that the Excursion Plate cannot be disabled by Redstone signals; they can only reverse the direction of the Plate's beam.
5. Security Tab
Allows setting who is allowed to configure the Excursion Plate. Only available if a Signalum Security Lock is installed.
