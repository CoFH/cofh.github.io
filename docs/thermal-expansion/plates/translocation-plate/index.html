--- title: Translocation Plate ---

The Translocation Plate is one of the Plates added by Thermal Expansion. When touched by entities, it teleports them a configurable amount of blocks away from the Plate in a certain direction. This works even when there are other blocks in between; in other words, the Translocation Plate works through walls. However, it doesn't work if the destination itself is blocked.

The Plate can be placed on any side of a block. Its initial direction depends on which part of the face the player is aiming at when placing the Plate. For example, aiming at the middle of the face results in a Plate pointed away from the block the Plate is placed on, and aiming slightly to the west results in a Plate pointed to the west.

The direction of the Plate can be changed using a Crescent Hammer or similar, as well as in the Plate's GUI.

Crafting


Fluid Transposer
  • Plate Frame
  • 1000 mB Resonant Ender

GUI

The Translocation Plate can be configured by sneaking and using a Multimeter on it. The Plate's GUI is then opened:

1. Distance Controls
Change how far away in blocks entities are teleported on contact. Can be anywhere between 0 and 16 blocks.
2. Direction Button
Changes the direction in which the Translocation Plate is pointing.
3. Information Tab
Displays a bit of information about the Translocation Plate.
4. Security Tab
Allows setting who is allowed to configure the Translocation Plate. Only available if a Signalum Security Lock is installed.
