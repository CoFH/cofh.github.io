--- title: BattleWrench ---
90% hammer, 10% wrench.

The BattleWrench is a tool/weapon added by Thermal Expansion. It is a combination of a Crescent Hammer and an Invar Sword, meaning it has the properties of both.

The item will take damage when used to attack an entity (but not when used as a wrench). Without enchantments, this can be done 450 times before it breaks. Like an Invar Sword, the BattleWrench can be repaired in an Anvil using Invar Ingots.

Crafting


Shaped Crafting
  • Crescent Hammer
  • Invar Gear
