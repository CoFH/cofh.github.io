--- title: Sawmill ---
Like punching trees, but better!

The Sawmill is one of the Machines added by Thermal Expansion. It uses Redstone Flux to cut wooden objects. It is capable of processing Wood logs more efficiently than by hand, as well as turning various things made of wood back into Wood Planks. The machine often produces Sawdust as a byproduct.

Crafting

  • Machine Frame (any tier)
  • Redstone Reception Coil
  • 2x Copper Gear
  • 2x Wood Planks (any type)
  • Iron Axe

Power

When working at maximum speed, the Sawmill consumes 20 RF per tick. When its internal energy storage is starting to deplete, the Machine will slow down and use less power. This prevents sudden interruptions of the Sawmill's work when its power supply cuts short.

GUI

1. Input Slot
Items placed in this slot are sawed by the Sawmill.
2. Primary Output Slots
Fully processed items end up in these slots.
3. Secondary Output Slot
Possible byproducts of items being sawed end up in this slot.
4. Progress Arrow
Displays the progress of the item currently being processed. If Not Enough Items is installed, the arrow can be clicked to look up the Sawmill's recipes.
5. Power Gauge
Displays how fast the Sawmill is working in relation to its maximum speed.
6. Energy Gauge
Displays how much Redstone Flux is currently stored in the Sawmill.
7. Charging Slot
Items that store Redstone Flux that are placed here are discharged by the Sawmill to charge itself.
8. Energy Tab
Displays how much power the Sawmill uses, and how much Redstone Flux is currently stored in it.
9. Information Tab
Displays a bit of information about the Sawmill.
10. Tutorial Tab
Explains various things about the Sawmill and the other tabs in the GUI.
11. Augmentation Tab
Augments can be placed in the slots in this tab.
12. Security Tab
Allows setting who is allowed to access the Sawmill. Only available if a Signalum Security Lock is installed.
13. Redstone Control Tab
Allows setting how the Sawmill responds to Redstone signals. Only available if the Integrated Redstone Circuit Augment is installed.
14. Configuration Tab
Allows configuring the input/output behavior of the sides of the Sawmill. Only available if the Integrated Modular Framework Augment is installed.
15. Player Inventory
The inventory and hotbar of the player that is accessing the Sawmill.

Input/Output

The different sides of the Sawmill correspond to different color-coded slots in its GUI. These sides can be identified by color-coded holes on the sides of the Sawmill. This is used for handling input and output with ducts/pipes/conduits/tubes, other Machines and other types of inventories.

If the Integrated Modular Framework Augment is installed, this feature of the Sawmill can be configured. Some things can be inserted from any side.

If the Integrated Servo Mechanism Augment is installed, output items are automatically ejected from the corresponding sides of the Sawmill, evenly divided between the available outputs.

Type Color Default Sides
Item input Blue Top
Primary item output Red Left, Right, Back
Secondary item output Yellow Bottom
Any item output Orange (not set)
Any item input/output* (colorless) (not set)
Redstone Flux input - (any side)
*: Cannot be automatically ejected from.
