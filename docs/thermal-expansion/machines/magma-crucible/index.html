--- title: Magma Crucible ---
Yes, it can make lava. You monster.

The Magma Crucible is one of the Machines added by Thermal Expansion. It uses Redstone Flux to melt down solid materials into fluids. This often takes a large amount of energy. The Machine is required to create most of the fluids in Thermal Foundation.

Crafting


Shaped Crafting
  • Machine Frame (any tier)
  • Redstone Reception Coil
  • 2x Invar Gear
  • 2x Nether Brick (block)
  • Leadstone Energy Cell Frame

Power

When working at maximum speed, the Magma Crucible consumes 400 RF per tick. When its internal energy storage is starting to deplete, the Machine will slow down and use less power. This prevents sudden interruptions of the Magma Crucible's work when its power supply cuts short.

The Magma Crucible is currently the Machine in Thermal Expansion that can consume the most power, consuming up to 8000 RF per tick with all three Processing Speed Augments installed.

GUI

1. Input Slot
Items placed in this slot are molten down by the Magma Crucible.
2. Output Tank
Fluids resulting from molten down items end up in this tank.
3. Progress Arrow
Displays the progress of the item currently being processed. If Not Enough Items is installed, the arrow can be clicked to look up the Magma Crucible's recipes.
4. Power Gauge
Displays how fast the Magma Crucible is working in relation to its maximum speed.
5. Energy Gauge
Displays how much Redstone Flux is currently stored in the Magma Crucible.
6. Charging Slot
Items that store Redstone Flux that are placed here are discharged by the Magma Crucible to charge itself.
7. Energy Tab
Displays how much power the Magma Crucible uses, and how much Redstone Flux is currently stored in it.
8. Information Tab
Displays a bit of information about the Magma Crucible.
9. Tutorial Tab
Explains various things about the Magma Crucible and the other tabs in the GUI.
10. Augmentation Tab
Allows installing Augments in the Machine.
11. Security Tab
Allows setting who is allowed to access the Magma Crucible. Only available if a Signalum Security Lock is installed.
12. Redstone Control Tab
Allows setting how the Magma Crucible responds to Redstone signals. Only available if the Integrated Redstone Circuit Augment is installed.
13. Configuration Tab
Allows configuring the input/output behavior of the sides of the Magma Crucible. Only available if the Integrated Modular Framework Augment is installed.
14. Player Inventory
The inventory and hotbar of the player that is accessing the Magma Crucible.

Input/Output

The different sides of the Magma Crucible correspond to different color-coded slots and tanks in its GUI. These sides can be identified by color-coded openings in the sides of the Magma Crucible. This is used for handling input and output with ducts/pipes/conduits/tubes, other Machines and other types of inventories and/or fluid tanks.

If the Integrated Modular Framework Augment is installed, this feature of the Magma Crucible can be configured. Some things can be inserted from any side.

If the Integrated Servo Mechanism Augment is installed, output fluids are automatically ejected from the corresponding sides of the Magma Crucible, evenly divided between the available outputs.

If the Integrated Hopper Apparatus Augment is installed, input items are automatically pulled into the Magma Crucible from adjacent inventories at the corresponding sides.
