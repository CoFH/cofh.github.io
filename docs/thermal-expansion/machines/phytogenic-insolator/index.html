--- title: Phytogenic Insolator ---
Definitely not organic.

The Phytogenic Insolator is one of the Machines added by Thermal Expansion. It uses Redstone Flux to infuse plants with 'sunlight', Water and fertilizer. The Machine can be used to grow plants at a steady rate and in an organized way.

Because simulating sunlight takes a lot of energy and growing plants still takes some time, the Phytogenic Insolator is somewhat slower than most other Machines in the mod.

Crafting


Shaped Crafting
  • Machine Frame (any tier)
  • Redstone Reception Coil
  • 2x Copper Gear
  • 2x Dirt
  • Lumium Gear

Power

When working at maximum speed, the Phytogenic Insolator consumes 20 RF per tick. When its internal energy storage is starting to deplete, the Machine will slow down and use less power. This prevents sudden interruptions of the Phytogenic Insolator's work when its power supply cuts short.

GUI

1. Primary Input Slot / Fertilizer Slot
Items placed in this slot are used together with items placed in the next slot to grow plants. Can be locked so that only fertilizer-type items can enter this slot. Currently the only items that can be used as fertilizer are Phyto-Gro and Rich Phyto-Gro.
2. Secondary Input Slot
Items placed in this slot are used together with items placed in the previous slot to grow plants.
3. Lock/Unlock Fertilizer Slot Button
This button can be clicked to lock the primary input slot / fertilizer slot to only accept fertilizer-type items (see 1).
4. Water Tank
Stores Water that is to be used to grow plants.
5. Primary Output Slot
Fully grown plants end up in this slot.
6. Secondary Output Slot
Possible byproducts of plants being grown (mostly seeds) end up in this slot.
7. Progress Arrow
Displays the progress of the plant currently being grown. If Not Enough Items is installed, the arrow can be clicked to look up the Insolator's recipes.
8. Power Gauge
Displays how fast the Insolator is working in relation to its maximum speed.
9. Energy Gauge
Displays how much Redstone Flux is currently stored in the Insolator.
10. Charging Slot
Items that store Redstone Flux that are placed here are discharged by the Insolator to charge itself.
11. Energy Tab
Displays how much power the Insolator uses, and how much Redstone Flux is currently stored in it.
12. Information Tab
Displays a bit of information about the Insolator.
13. Tutorial Tab
Explains various things about the Insolator and the other tabs in the GUI.
14. Augmentation Tab
Augments can be placed in the slots in this tab.
15. Security Tab
Allows setting who is allowed to access the Insolator. Only available if a Signalum Security Lock is installed.
16. Redstone Control Tab
Allows setting how the Insolator responds to Redstone signals. Only available if the Integrated Redstone Circuit Augment is installed.
17. Configuration Tab
Allows configuring the input/output behavior of the sides of the Insolator. Only available if the Integrated Modular Framework Augment is installed.
18. Player Inventory
The inventory and hotbar of the player that is accessing the Insolator.

Input/Output

The different sides of the Phytogenic Insolator correspond to different color-coded slots in its GUI. These sides can be identified by color-coded holes on the sides of the Insolator. This is used for handling input and output with ducts/pipes/conduits/tubes, other Machines and other types of inventories.

If the Integrated Modular Framework Augment is installed, this feature of the Phytogenic Insolator can be configured. Some things can be inserted from any side.

If the Integrated Servo Mechanism Augment is installed, output items are automatically ejected from the corresponding sides of the Phytogenic Insolator, evenly divided between the available outputs.

If the Integrated Hopper Apparatus Augment is installed, input items are automatically pulled into the Phytogenic Insolator from adjacent inventories at the corresponding sides.

Recipes

The table below lists the recipes that the Phytogenic Insolator can perform. Recipes that use less energy also take less time.

Recipes always consist of a type of plant or seed, a fertilizer item and Water. Either Phyto-Gro or Rich Phyto-Gro must be used as fertilizer. With Phyto-Gro, plants take 7200 RF to grow. With Rich Phyto-Gro, however, both the primary output amount and the secondary output chance are multiplied by 3 (configurable), but plants take 9600 RF to grow.

Without Augments, the amount of Water used in a recipe is equal to the amount of energy used divided by 10, but the amount decreases if Machine Speed Augments are installed.

*: May be changed or disabled in the config.
**: Chance doesn't change when using Rich Phyto-Gro.
