--- title: "Dynamo: Side Accessibility" search_headers: - title: 'Transmission Coil Ducting' ---

Dynamo: Side Accessibility (seen ingame as the Transmission Coil Ducting) is a type of Augment that can be installed in Dynamos. It allows items and fluids to be inserted and extracted through a Dynamo's transmission coil: the side that outputs generated Redstone Flux.

This Augment type is useful for creating compact setups with Dynamos; especially when Fluctuating Itemducts and/or Flux-Plated Fluiducts are added to the mix.
