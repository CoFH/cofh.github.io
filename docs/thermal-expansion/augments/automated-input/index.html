--- title: Automated Input search_headers: - title: 'Integrated Hopper Apparatus' ---

Automated Input (seen ingame as the Integrated Hopper Apparatus) is a type of Augment that allows Machines to automatically transfer input items into themselves, pulling from adjacent inventories. An evenly divided amount of items is pulled from all available inventories.

Items are transferred in every 1.6 seconds (32 ticks). The amount of items transferred at once is limited by Machine Frame tier: Basic Machines can transfer 8 items at a time, Hardened Machines 16 items, Reinforced Machines 32 items and Resonant Machines 64 items.

Unlike Automated Output, this Augment type does not work with fluids.
