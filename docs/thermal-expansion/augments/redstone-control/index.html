--- title: Redstone Control ---

Redstone Control (seen ingame as the Integrated Redstone Circuit) is a type of Augment that allows devices to react to Redstone signals. When installed, it adds the Redstone Control tab to a device's GUI.

This tab allows configuring how a device reacts to Redstone signals. By default, devices ignore Redstone signals and are active whenever possible. With this Augment installed, they can be configured to only be active when powered or not powered by Redstone.

Newly crafted devices come with this Augment already installed, unless disabled in the config.
